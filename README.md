# BookStore Frontend

ReactJS Frontend for BookStore E-Commerce Application.

## Features

* Responsive UI
* Product Search
* Product Details
* Shopping Cart
* Wishlist
* Order History
* Customer Profile
* UPI Payment Page
* Admin Dashboard

## Tech Stack

* ReactJS
* React Router
* Axios
* Bootstrap
* React Icons

## Run Locally

npm install

npm start
